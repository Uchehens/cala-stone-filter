README

A simple word filter program for:-

– filter out all the words that contains a vowel in the middle of the word 
– filter out words that have length less than 3
– filter out words that contains the letter ‘t’

And then display the resulting text on the console.


Project Executions:
1. Clone the app and build using Visual Studio IDE
2. Hit run or Ctrl + F5 to run
3. Enter a valid test input file path then hit enter
